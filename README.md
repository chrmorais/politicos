Database of political entities

## Install


1.  Clone the repository:

        git clone https://github.com/olhoneles/politicos.git

1.  Create a *virtualenv*:

        cd politicos
        virtualenv .

1.  Install dependencies

        make setup

1.  Create your database

         make data

1.  Run it

        make run

1.  Run tests

        make test
