Django + Tastypie + Swagger

A base da Politicos API é o framework Django. Por isso optamos em usar o Tastypie, que é um framework Python poderoso e altamente personalizável de criação de APIs. Também usamos o Swagger, que é uma representação simples mas poderosa de uma API RESTful.

O endereço da API é:

http://politicos.olhoneles.org/api/v0/

Você pode ver todas as rotas da API no item Demonstração no menu superior.

Alguns exemplos de rotas:

http://politicos.olhoneles.org/api/v0/educations/
http://politicos.olhoneles.org/api/v0/candidacies/?election_round__election__year=2014

Software Livre

O código da API é livre e licenciado sob a Affero GPL. Você pode pegar o código executando o seguinte comando:

git clone https://github.com/olhoneles/politicos.git

Contribuindo

Vamos deixar as contribuições para este projeto tão fácil e transparente quanto possível. Portanto faça um fork do repositório e nos envie suas melhorias.

Usamos o GitHub para armazenar o código, acompanhar as questões e solicitações de recursos.

Ajuda

Você pode abrir um ticket via GitHub, mas se preferir temos uma lista de email do projeto e também estamos online via IRC no canal #olhoneles na Freenode.
